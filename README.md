# Student Management System

## Description
A simple Student Management System developed using Python and Object-Oriented Programming (OOP) concepts. The program stores and displays student details such as Name, Roll Number, Student ID, Marks, and Grade.

## Features
- Add multiple students
- Store student records
- Automatic grade calculation
- Display student information

## Technologies Used
- Python

## Concepts Used
- Classes and Objects
- Constructors
- Methods
- Conditional Statements
- Lists
- User Input Handling
